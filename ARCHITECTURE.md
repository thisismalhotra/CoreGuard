# Architectural Specification (ASD): Core-Guard MVP **Tech Stack:** Next.js (Frontend), Python FastAPI (Backend), PostgreSQL (DB), Pinecone (Vector), OpenAI (LLM).  ## 1. High-Level Architecture The system follows an **Event-Driven Microservices** pattern, simulated locally for the MVP.  `[Next.js Dashboard]` <--> `[Socket.io]` <--> `[Python FastAPI Orchestrator]` <--> `[Agents (LangChain)]`  * **Frontend:** Visualizes state. Emits simulation triggers. * **Backend API:** Handles CRUD operations for Inventory/Orders. * **Agent Orchestrator:** Manages the state machine and context for LLM agents. * **Database:** Stores structured operational data (SQL) and unstructured engineering specs (Vector).  ## 2.
